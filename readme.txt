Un formulario básico para registrar reclamaciones según las normativas de Perú.

== Descripción ==

Este plugin añade un formulario de reclamaciones compatible con las normativas peruanas. Los usuarios podrán enviar reclamaciones desde su sitio web, y la empresa recibirá las quejas directamente por correo electrónico. El plugin incluye la opción de seleccionar departamentos, provincias y distritos, y está diseñado para facilitar la gestión de las reclamaciones de clientes.

== Características principales ==
* Formulario personalizado para registrar reclamaciones según las normativas peruanas.
* Sistema de selección de departamento, provincia y distrito.
* Envío de notificaciones por correo electrónico a la empresa.
* Shortcode fácil de usar: `[libro_reclamaciones]` para insertar el formulario en cualquier página o entrada.
* Personalización de ajustes desde el panel de administración.

== Instalación ==

1. Descarga y extrae el archivo zip del plugin.
2. Sube la carpeta `libro-reclamaciones-peru` al directorio `/wp-content/plugins/`.
3. Activa el plugin desde el menú 'Plugins' en WordPress.
4. Inserta el shortcode `[libro_reclamaciones]` en cualquier página o entrada para mostrar el formulario.

== FAQ ==

= ¿Cómo inserto el formulario en una página o entrada? =
Usa el shortcode `[libro_reclamaciones]` donde quieras que aparezca el formulario.

= ¿Puedo personalizar los campos del formulario? =
Actualmente, el formulario está configurado según las normativas de Perú. Sin embargo, puedes modificar el archivo `form-template.php` para hacer ajustes adicionales.

= ¿Recibiré una notificación por cada reclamo? =
Sí, el plugin envía una notificación por correo electrónico cada vez que se registra un nuevo reclamo.
